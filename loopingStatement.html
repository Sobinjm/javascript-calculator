Looping Statement

What is Looping Statement?

Looping statement is used to repeat a block of code multiple times.

For Loop

For loop is used to repeat a block of code a specific number of times.

For Loop Syntax

for (initialization; condition; increment/decrement) {
    statement(s) - codeblock
}

For Loop Example

for (let i = 0; i < 5; i++) {
    console.log(i); //consoles number 0-4
}

While Loop

While loop is used to repeat a block of code as long as a condition is true.

While Loop Syntax

while (condition) {
    statement(s)
}

While Loop Example

let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}
